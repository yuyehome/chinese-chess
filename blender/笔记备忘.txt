【】棋子制作核心流程

	【】做低模，展UV。
		【】棋子直径40mm，厚度14mm，倒角width 4mm , segments 3。

	【】做高模。
		【】复制一份低模出来。可考虑增加细分，也可以不加。。。
		【】外圈圆环凹槽，宽1mm，深2mm，记得也要倒角。
		【】blender里面打字， a标签中修改属性，size 0.03，居中对齐。
		【】汉字转成网络，有限隔并，去掉多余的线，如果空了没有面了，再填充，再次有限隔并。
		【】汉字E挤出2mm的厚度。与高模做布尔差集，应用。
		【】镂空的只选择凹槽底部进行倒角，倒角数据0.0005, seg 3 。顶面不倒角，因为特麻烦。
		【】【】【】如果发生奇怪的光影，则增加两个修改器 bevel （amount 0 ）和weighted normal，即可正常

	【】烘焙。cycles，低模新建纹理，先选低模再选高模，再烘焙。

为低模设置烘焙目标 (Target)：
选中我们的 Chess_LowPoly 模型。
切换到顶部的 "Shading" (着色) 工作空间。
在底部的材质节点编辑器中，确保 Chess_LowPoly 有一个材质。如果没有，点击**“新建”**。
按 Shift + A，在弹出的菜单中选择 纹理 -> 图像纹理 (Texture -> Image Texture)，在节点编辑器中新建一个图像纹理节点。
点击这个新节点上的**“新建”**按钮，会弹出一个窗口来创建新图像：
名称 (Name): Normal_Shuai
宽度 (Width) & 高度 (Height): 2048 x 2048 像素。这个分辨率足够精细。
颜色 (Color): 保持默认黑色即可，无所谓。
勾选“32位浮点” (32-bit Float): 非常重要！ 这能提供更高的精度，防止法线贴图出现色阶断层。
点击**“确定”**。
确保这个新建的“图像纹理”节点是“活动”状态（它会有一个白色的高亮边框）。Blender会把烘焙结果输出到当前活动的图像纹理节点上。
最后一步关键设置： 在这个 Normal_Shuai 节点上，将**“色彩空间” (Color Space)** 从 sRGB 改为 非彩色 (Non-Color)。因为法线贴图存储的是方向数据，而不是颜色信息。

配置烘焙参数
回到**“渲染属性”**（小相机图标）面板。
找到并展开最下方的**“烘焙” (Bake)** 选项卡。
进行以下设置：
烘焙类型 (Bake Type): 法线 (Normal)。
空间 (Space): 切向 (Tangent)。这是游戏引擎使用的标准法线空间。
勾选“从选中项到活动项” (Selected to Active): 这是烘焙的核心。它告诉Blender，我们要将“选中项”（Selected，即高模）的细节，烘焙到“活动项”（Active，即低模）上。
展开**“从选中项到活动项”**的下拉菜单，设置：
挤出 (Extrusion): 设置一个较小的值，比如 1mm。这个值代表从低模表面向外发射光线的“起点”，用于捕捉那些比低模略大的高模细节。
最大光线距离 (Max Ray Distance): 设置为 0 即可。在较新版本的Blender中，挤出值已经能很好地处理距离问题。如果烘焙出的贴图有黑斑，再尝试将这个值也设为 1mm。

步骤2：为凹槽上色 (核心技巧)
我们要实现的是：棋子主体是木纹，但凹槽里是纯色（红/黑/蓝/绿）。我们将通过烘焙一张**“遮罩贴图” (Mask Map)** 来实现。
烘焙“环境光遮蔽” (Ambient Occlusion, AO) 贴图：
AO贴图是一种灰度图，它会计算模型缝隙、角落等光线难以到达的地方，并将其变暗。模型的凹槽在AO图里会自然地变成深色，这正是我们需要的完美“选区”。
在材质节点中，新建一个**“图像纹理”**节点，创建一个新的 2048x2048 图像，命名为 Chess_AO (无需32位浮点，8位即可，色彩空间sRGB没问题)。确保它是活动节点。
进入**“渲染属性”**面板 -> “烘焙”。
烘焙类型改为**“环境光遮蔽” (Ambient Occlusion)**。
这次不要勾选“从选中项到活动项”！ 我们是基于低模自身来计算AO的。
点击**“烘焙”**。完成后，你会得到一张灰度图，其中文字和圆圈凹槽是深灰色/黑色，其余部分是白色。
保存这张 Chess_AO.png 图片！
在节点编辑器中使用AO贴图作为遮罩：
现在我们的节点网络会变得有趣起来。我们需要一个节点，能根据AO贴图的黑白信息，在“木纹颜色”和“凹槽颜色”之间进行选择。这个节点就是**“混合颜色” (Mix Color)**。
按 Shift+A -> 颜色 -> 混合颜色，将它放在“色相/饱和度”节点和“原理化BSDF”的“基础颜色”输入之间。
将你的木纹颜色（从“色相/饱和度”节点输出）连接到“混合颜色”节点的 B 输入口。
在 A 输入口，设置你想要的凹槽颜色，比如红色。
将我们刚刚烘焙好的 Chess_AO.png 图像纹理节点拖进来，将其“颜色”输出连接到“混合颜色”节点的**“系数” (Factor)** 输入口。
微调遮罩： 通常AO贴图的对比度不够强。在AO节点和“系数”输入口之间，再加一个 Shift+A -> 颜色 -> 颜色渐变 (Color -> Color Ramp) 节点。拖动颜色渐变的黑白滑块，增加对比度，直到你看到凹槽被精准地填充上了红色，而其他区域保持木纹。
